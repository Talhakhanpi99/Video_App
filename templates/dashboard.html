Runtime Secrets

Runtime secrets are values like your Gemini API key or Pexels API key. They are saved after install inside the app's private storage on the phone, instead of being bundled openly inside the APK files.

How to use them:

1. Open a tool that needs internet. 2. Save your key once. 3. The app reads it from private runtime storage on later launches.

Important: this is safer than shipping keys inside the APK, but it is still not perfect protection against a rooted device or reverse engineering.
